📚✨ Student Management System (C++)

🚀 The Student Management System is a console-based application developed in C++ designed to efficiently manage and organize student academic records. The project simulates a lightweight database system and is built to strengthen understanding of core programming concepts such as data structures, file handling, algorithms, and modular software design.

🎯 Overview

📌 Each student record consists of:

🆔 Unique ID
👤 Full Name
📊 Grades for 5 subjects

The system stores and manipulates data dynamically using C++ vectors, allowing efficient memory management and scalability.

⚙️ Core Features

 ➕ Add new student records
 📋 Display all students in a structured table format
 🔍 Search students by ID or name
 ✏️ Update existing student information
 ❌ Delete student records safely

These operations replicate fundamental CRUD (Create, Read, Update, Delete) database functionality.

📈 Performance Analysis

⭐ The system includes an academic evaluation module:

📉 Calculates average grade for each student
🏆 Identifies top-performing students
📊 Sorts students in descending order based on performance

This provides a clear and structured overview of student rankings and progress.

💾 Data Persistence

💽 The project ensures data is not lost after execution using file handling (fstream):

📂 Saves all records to an external file (Data.txt)
📥 Automatically loads existing data on startup
🔄 Maintains continuity between sessions
🖥️ User Interface

🧭 The system is built with a menu-driven interface, ensuring:

🧩 Simple navigation between options
🛡️ Input validation for data integrity
📌 Clear and structured user interaction
🚀 Educational Purpose

🎓 This project is designed to reinforce:

🧠 C++ programming fundamentals
🏗️ Problem-solving and system design
📂 File handling and data storage techniques
🔢 Algorithmic thinking and sorting logic
